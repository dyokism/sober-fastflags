## Что такое Sober?

[Sober](https://sober.vinegarhq.org/) — это слой совместимости, запускающий Android-приложение Roblox (APK) на рабочих столах Linux. Он распространяется в формате Flatpak (`org.vinegarhq.Sober`) и использует Vulkan в качестве основного графического API (с OpenGL в качестве запасного). Конфигурация находится в `~/.var/app/org.vinegarhq.Sober/config/sober/config.json`. Вы также можете открыть графическое меню настроек с помощью команды `flatpak run org.vinegarhq.Sober config` или нажав правой кнопкой мыши по Sober в меню приложений и выбрав **Settings**.

## Что такое FastFlags?

FastFlags (FFlags) — это внутренние переменные движка Roblox, которые управляют рендерингом, интерфейсом, стабильностью и многим другим. С **29 сентября 2025 года** Roblox применяет строгий белый список (allowlist) — только небольшой набор флагов может быть переопределен локально через конфигурационные файлы. Любой флаг, отсутствующий в списке, игнорируется клиентом без каких-либо уведомлений.

> [!IMPORTANT]
> Это руководство охватывает только те флаги, присутствие которых в текущем белом списке подтверждено. Флаги из старых руководств сообщества могут больше не работать.

## Подтвержденные активные FFlags

### Рендеринг и производительность

Эти флаги управляют детализацией геометрии, сглаживанием, освещением и дальностью прорисовки травы.

| Имя флага | Тип | Диапазон значений | Описание работы |
| :--- | :--- | :--- | :--- |
| `DFIntCSGLevelOfDetailSwitchingDistance` | int | `0` - `1000` | Основное расстояние отсечения LOD для CSG-моделей. Меньше = выше FPS. |
| `DFIntCSGLevelOfDetailSwitchingDistanceL12` | int | `0` - `1000` | Расстояние LOD для качества графики 1–2. |
| `DFIntCSGLevelOfDetailSwitchingDistanceL23` | int | `0` - `1000` | Расстояние LOD для качества графики 2–3. |
| `DFIntCSGLevelOfDetailSwitchingDistanceL34` | int | `0` - `1000` | Расстояние LOD для качества графики 3–4. |
| `FIntDebugForceMSAASamples` | int | `1`, `2`, `4` | Принудительно включает сглаживание MSAA (более сглаженные края, нагружает видеокарту). |
| `DFIntDebugFRMQualityLevelOverride` | int | `0` - `21` | Переопределяет ползунок уровня графики (позволяет выйти за рамки стандартных значений 1–10). |
| `FIntFRMMaxGrassDistance` | int | `0` - `1000` | Максимальная дальность прорисовки травы. Установите значение `0`, чтобы отключить ее. |
| `FIntFRMMinGrassDistance` | int | `0` - `1000` | Минимальное расстояние начала рендеринга травы. |

### Стабильность и видеопамять (VRAM)

Эти флаги помогают предотвратить вылеты из-за нехватки памяти (out-of-memory), особенно на видеокартах с ограниченным объемом видеопамяти (VRAM).

| Имя флага | Тип | Диапазон значений | Описание работы |
| :--- | :--- | :--- | :--- |
| `DFFlagTextureQualityOverrideEnabled` | bool | `true` / `false` | Включает ручное управление разрешением текстур. |
| `DFIntTextureQualityOverride` | int | `0` - `3` | Устанавливает качество текстур (0 = самое низкое, 3 = максимальное). |

> [!WARNING]
> Установка флага `DFIntTextureQualityOverride` в значение `3` на видеокартах с 4 ГБ видеопамяти или меньше с высокой вероятностью приведет к мгновенному вылету `RBXCRASH: OutOfMemory`. Используйте `2` или `1` для стабильности.

### Интерфейс и окружение

Второстепенные флаги, влияющие на визуальный комфорт и поведение интерфейса.

| Имя флага | Тип | Диапазон значений | Описание работы |
| :--- | :--- | :--- | :--- |
| `FIntGrassMovementReducedMotionFactor` | int | `0` - `1000` | Управляет интенсивностью анимации покачивания травы. `0` = полностью статична. |

## Предустановки конфигурации

Скопируйте один из пресетов ниже и вставьте его в конфигурационный файл Sober:

```
~/.var/app/org.vinegarhq.Sober/config/sober/config.json
```

### Предустановка 1: Низкие настройки / Исправление вылетов видеопамяти

Для видеокарт с объемом памяти менее 4 ГБ, интегрированной графики или систем, испытывающих вылеты из-за нехватки памяти (`OutOfMemory`).

```json
{
  "enable_hidpi": false,
  "fflags": {
    "DFFlagTextureQualityOverrideEnabled": true,
    "DFIntTextureQualityOverride": 1,
    "DFIntCSGLevelOfDetailSwitchingDistance": 100,
    "DFIntCSGLevelOfDetailSwitchingDistanceL12": 75,
    "DFIntCSGLevelOfDetailSwitchingDistanceL23": 100,
    "DFIntCSGLevelOfDetailSwitchingDistanceL34": 150,
    "FIntFRMMaxGrassDistance": 0,
    "FIntGrassMovementReducedMotionFactor": 0
  }
}
```

### Предустановка 2: Сбалансированная / Средний уровень

Для видеокарт среднего уровня (класс GTX 1650, RX 580) с 4–6 ГБ видеопамяти. Хороший баланс между визуальным качеством и производительностью.

```json
{
  "enable_hidpi": false,
  "fflags": {
    "DFFlagTextureQualityOverrideEnabled": true,
    "DFIntTextureQualityOverride": 2,
    "DFIntCSGLevelOfDetailSwitchingDistance": 400,
    "DFIntCSGLevelOfDetailSwitchingDistanceL12": 200,
    "DFIntCSGLevelOfDetailSwitchingDistanceL23": 350,
    "DFIntCSGLevelOfDetailSwitchingDistanceL34": 500,
    "FIntDebugForceMSAASamples": 2,
    "FIntFRMMaxGrassDistance": 200,
    "FIntGrassMovementReducedMotionFactor": 50
  }
}
```

### Предустановка 3: Максимальное качество графики

Для мощных систем с объемом видеопамяти 8 ГБ и более. Принудительно задает максимальную детализацию, сглаживание и качество текстур.

```json
{
  "enable_hidpi": true,
  "fflags": {
    "DFFlagTextureQualityOverrideEnabled": true,
    "DFIntTextureQualityOverride": 3,
    "DFIntCSGLevelOfDetailSwitchingDistance": 1000,
    "DFIntCSGLevelOfDetailSwitchingDistanceL34": 1000,
    "FIntDebugForceMSAASamples": 4,
    "DFIntDebugFRMQualityLevelOverride": 21
  }
}
```

> [!TIP]
> Вы можете комбинировать флаги из разных пресетов. Например, использовать расстояния LOD из «Сбалансированного» пресета с «Максимальным» качеством текстур, если ваша видеокарта имеет достаточно видеопамяти, но испытывает трудности с геометрией.

## Подробные темы

<details>
<summary><strong>Детализация (LOD) и масштабирование геометрии</strong></summary>

Карты Roblox часто используют сложные конструкции CSG (Constructive Solid Geometry). Рендеринг таких объектов на больших расстояниях создает серьезную нагрузку как на процессор, так и на видеокарту.

Снизив значение флага `DFIntCSGLevelOfDetailSwitchingDistance` (например, до `150`), вы заставите игру заменять сложные модели их низкополигональными аналогами ближе к камере. Это значительно увеличивает частоту кадров, не меняя физические хитбоксы столкновений — объекты ведут себя точно так же, но выглядят проще на расстоянии.

Версии флагов с суффиксами уровней (`L12`, `L23`, `L34`) позволяют настраивать это для каждого уровня графики отдельно, за счет чего при низких настройках отсечение происходит более агрессивно.

</details>

<details>
<summary><strong>Распределение видеопамяти (VRAM) и вылеты по нехватке памяти (OOM)</strong></summary>

Sober запускает Android-версию Roblox внутри окружения Linux Flatpak. Android-приложение рассчитывает на общую архитектуру памяти мобильного устройства, которая отличается от управления VRAM в драйверах настольного Linux.

**Проблема:** когда движок запрашивает текстуры максимального качества, он может быстро исчерпать выделенную VRAM видеокарты. В Windows драйвер перенес бы избыточные данные в системную оперативную память. В Linux (особенно с проприетарными драйверами NVIDIA) этот механизм работает нестабильно, что приводит к мгновенному вылету с ошибкой `RBXCRASH: OutOfMemory`.

**Решение:** установите флаг `DFFlagTextureQualityOverrideEnabled` в значение `true`, а `DFIntTextureQualityOverride` — в значение `2` (среднее) или `1` (низкое). Это заставит движок запрашивать у сервера текстуры меньшего разрешения, сохраняя потребление видеопамяти в безопасных пределах.

</details>

<details>
<summary><strong>Графические API: Vulkan против OpenGL</strong></summary>

Выбор графического API управляется конфигурацией Sober, **а не** внутренними флагами FFlags.

- По умолчанию Sober использует **Vulkan** для максимальной производительности.
- Если вы сталкиваетесь с артефактами, черным экраном или сбоями запуска (часто на старых GPU или гибридных ноутбуках), принудительно включите OpenGL, установив `"use_opengl": true` в корне `config.json`.

Не используйте флагов движка вроде `FFlagDebugGraphicsPreferVulkan` или `FFlagDebugGraphicsPreferOpenGL` — они вызывают сбои из-за несовпадения контекста.

</details>

<details>
<summary><strong>Замена ресурсов (кастомные текстуры и курсоры)</strong></summary>

Sober позволяет заменять игровые ресурсы через директорию `asset_overlay`:
`~/.var/app/org.vinegarhq.Sober/data/sober/asset_overlay`

Файлы из этой папки имеют приоритет перед стандартными ресурсами Roblox при перезапуске приложения. Структура папок повторяет `packages/*/com.roblox.client/base.apk/assets`.

Пример для кастомных курсоров мыши:
```
~/.var/app/org.vinegarhq.Sober/data/sober/asset_overlay
└── content
    └── textures
        └── Cursors
            └── KeyboardMouse
                ├── ArrowCursor.png
                ├── ArrowFarCursor.png
                └── IBeamCursor.png
```
Чтобы вернуть стандартные ресурсы, просто очистите папку `asset_overlay`.

</details>

<details>
<summary><strong>Полноэкранный режим (F11) и автозакрытие</strong></summary>

Переключатель полноэкранного режима в самом Roblox не работает на Android-версии. В Sober используйте клавишу `F11` для входа и выхода из полноэкранного режима. Sober запоминает это состояние.

Чтобы приложение автоматически закрывалось при выходе из игры, добавьте `"close_on_leave": true` в ваш `config.json`.

</details>

## Разблокировка частоты кадров (FPS)

FastFlag `DFIntTaskSchedulerTargetFps` больше не работает — он был удален из белого списка. Чтобы изменить ограничение частоты кадров, необходимо отредактировать файл настроек XML напрямую.

### Инструкция:

1. Запустите любой плейс Roblox через Sober для генерации конфигурационных файлов.
2. Полностью закройте клиент.
3. Перейдите по пути `~/.var/app/org.vinegarhq.Sober/data/sober/appData/`.
4. Откройте файл `GlobalBasicSettings_13.xml` в текстовом редакторе.
5. Найдите строку: `<int name="FramerateCap">60</int>`
6. Измените значение `60` на желаемую частоту кадров (например, `144`, `240`) или укажите `0`, чтобы снять ограничение.
7. Сохраните изменения, закройте редактор и запустите Roblox снова.

> [!NOTE]
> Перед редактированием этого файла обязательно закройте Roblox. Клиент перезаписывает этот файл при выходе, поэтому все изменения, внесенные во время работы игры, будут потеряны.

## Безопасность и риски античита

Roblox использует систему античита Hyperion (Byfron). Настройка разрешенных FFlags в файле `config.json` абсолютно безопасна. Попытки обойти белый список — нет.

> [!CAUTION]
> Следующие действия несут высокий риск получения постоянной блокировки аккаунта или бана по железу. Не пытайтесь их совершать.

- **Манипуляции с файлами кэша (`IxpSettings.json`):** внедрение неавторизованных флагов в файлы кэша и установка атрибута «только для чтения» распознается античитом Hyperion как вмешательство в файлы игры.
- **Редактирование памяти:** использование сторонних утилит для принудительной загрузки запрещенных флагов (например, изменение шага физического времени через `DFIntTimestepArbiterThresholdCFLThou` или обход текстур для валлхака) приводит к автоматическому бану.
- **Программы обхода белого списка:** любые программы или аргументы запуска, созданные с целью обхода фильтра FFlag, классифицируются как эксплойт.

## Устаревшие FFlags

Следующие флаги часто встречаются в старых инструкциях, но они больше не входят в белый список. Их добавление в `config.json` ни на что не влияет — клиент просто их игнорирует.

| Флаг | Причина удаления |
| :--- | :--- |
| `DFIntTaskSchedulerTargetFps` | Заменен редактированием `GlobalBasicSettings_13.xml`. |
| `FFlagTaskSchedulerLimitTargetFpsTo2402` | Удален из белого списка. |
| `DFFlagDebugPauseVoxelizer` | Блокировка воксельного освещения запрещена текущим белым списком. |
| `FFlagDebugSkyGray` | Серый скайбокс заблокирован текущим белым списком. |
| `DFIntConnectionMTUSize` | Флаги сетевой настройки заблокированы. |
| `FFlagDebugDisableTelemetryEphemeralCounter` | Блокировка отправки телеметрии запрещена. |
| `FFlagAdServiceEnabled` | Отключение рекламы заблокировано. |
| `FFlagMovePrerender` | Флаги управления потоками заблокированы. |
| `DFIntDebugDynamicRenderKiloPixels` | Масштабирование разрешения отклонено инженерами Roblox. |
